OpenAccountants is AGPL-3.0. AGPL-3.0, MIT, and LGPL-3.0 are all compatible licenses. Projects below can be incorporated with attribution.

## phpcfdi/cfdi-sat-scraper

- Repository: [phpcfdi/cfdi-sat-scraper](https://github.com/phpcfdi/cfdi-sat-scraper)
- License: MIT
- Stars: 93
- Language: PHP
- Scope: PHP library for web scraping the SAT CFDI download page. Used to programmatically download CFDI XML invoices from SAT's portal.
- Integration: MIT — fully compatible. Reference for SAT portal authentication, CFDI metadata extraction, and XML download workflows. Useful for validating CFDI document processing logic.

## FacturaScripts

- Repository: [NeoRazorX/facturascripts](https://github.com/NeoRazorX/facturascripts)
- License: LGPL-3.0
- Stars: 469
- Language: PHP / Spanish
- Scope: Open source ERP software built on modern PHP and Bootstrap. Includes Mexican CFDI module for facturación electrónica.
- Integration: LGPL-3.0 — compatible. Reference for Mexican CFDI generation, timbrado workflows, and facturación electrónica compliance. ERP-level tax computation patterns useful for validating IVA and ISR calculations.

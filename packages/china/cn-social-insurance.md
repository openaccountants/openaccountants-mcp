# 中国 — 社会保险与住房公积金（五险一金）— 技能 v1.0

> **范围说明：** 本技能是中国大陆按月计提与缴纳"五险一金"的端到端编排器。本技能不重复定义个人所得税税率表 —— 个税口径在 `china-pit` 中规定，本技能在工资税前扣除环节将"个人承担的五险一金"作为"专项扣除"对接至 `china-pit`。本技能给出**全国通用框架**，但中国五险一金的**缴费比例、基数上下限、调整时点**均由"参保地"市级（直辖市/地级市/计划单列市）人力资源和社会保障局、医保局、住房公积金管理中心分别规定，**差异极大**。任何具体数值必须以"当月、当地、最新公告"为准。
>
> **税收年度说明：** 中国社保与公积金按**自然月**缴纳；社保缴费基数大多于**每年7月**（部分地区1月）调整，参考上一年度当地"全口径城镇单位就业人员平均工资"；公积金缴存基数大多于**每年7月**调整。本技能所称"2025年"指**2025自然年**，涉及上海2024年度社平工资口径调整后的2024.7—2025.6缴费期间。
>
> **2025年重大政策提示：** 国务院**渐进式延迟法定退休年龄**方案自 **2025年1月1日**起实施 ——
> - 男职工：60岁 → 逐步延至 **63岁**；
> - 女干部（原55岁）：→ 逐步延至 **58岁**；
> - 女工人（原50岁）：→ 逐步延至 **55岁**；
> - 同时**最低缴费年限**自2030年起由15年逐步提高至20年。
>
> 上述变化影响养老缴费规划，但**不直接改变月度缴费比例**。本技能后续示例仍按月度比例编制。

下表以**上海市 2024.7—2025.6 缴费期间**为例（最常用的口径之一）。**北京、深圳、广州、杭州、成都等地比例差异显著**，使用前必须核对参保地最新公告。

|---|---|---|---|---|

| 是否签订有**社保互免协定**（外籍员工） | HR 入职审查 | 见第九节 |

## 三、养老保险（基本养老 — 企业职工）

### 3.1 法律依据

- 《**中华人民共和国社会保险法**》（2010年通过，2018年修订）第二章；
- 《**国务院关于建立统一的企业职工基本养老保险制度的决定**》（国发〔1997〕26号）；
- 《**国务院关于印发实施渐进式延迟法定退休年龄办法的通知**》（自 **2025.1.1** 起施行）；
- 各省/市《企业职工基本养老保险条例 / 实施办法》。

### 3.2 缴费比例

| 缴费主体 | 比例 | 备注 |
|---|---|---|
| **单位** | **16%** | 全国统一（自 2019.5.1 起由 20% 降至 16%）；部分地区曾有更低过渡 |
| **个人** | **8%** | 全国统一 |

> **个人 8% 全额计入个人账户**；单位 16% 计入统筹基金。

### 3.3 缴费基数

- **基数 = 员工上一年度月平均工资**（工资总额口径，含基本、奖金、津贴、补贴、加班费）；
- **下限 = 上一年度当地社平工资 × 60%**；
- **上限 = 上一年度当地社平工资 × 300%**；
- 上海 2024.7—2025.6 期间：下限约 **¥7,310/月**，上限约 **¥36,549/月**（仅示例）；
- 新参保员工：以**入职月工资**为初始基数；
- 基数确定后**一年不变**，下次7月调整。

### 3.4 计算

```
养老（单位）= 缴费基数 × 16%
养老（个人）= 缴费基数 × 8%   （从工资中代扣）
```

### 3.5 待遇与缴费年限

- **最低缴费年限**：达到法定退休年龄时，累计缴费年限 ≥ **15年**（2030年起逐步提至20年）方可按月领取基本养老金；
- 不足年限：可一次性补缴（部分地区已停止）或转为城乡居民养老；
- 待遇 = 基础养老金 + 个人账户养老金 + 过渡性养老金（视参保历史）。

## 四、医疗保险（含生育合并后）

### 4.1 法律依据

- 《社会保险法》第三章（基本医疗保险）、第六章（生育保险）；
- 《**国务院办公厅关于全面推进生育保险和职工基本医疗保险合并实施的意见**》（国办发〔2019〕10号）—— 自 2019.12 起，**生育保险并入医疗保险统一征收**，但生育待遇独立保留；
- 《**关于建立健全职工基本医疗保险门诊共济保障机制的指导意见**》（国办发〔2021〕14号）。

### 4.2 缴费比例（含合并后的生育保险）

| 缴费主体 | 比例（示例：上海 2024.7—2025.6） | 备注 |
|---|---|---|
| **单位** | **9.5%** | 含原生育保险 0.5%-1%；北京 9.8%、深圳 5.2%-6.7% |
| **个人** | **2%** | 全国普遍 2%；个人不缴生育保险 |
| **大病医疗补充**（部分地区） | 单位 1% 或固定金额 | 上海无单独大病项；北京另缴大病 |

### 4.3 缴费基数

同养老保险基数（口径一致 — 上下限同步）。

### 4.4 计算

```
医疗（单位）= 缴费基数 × 9.5%
医疗（个人）= 缴费基数 × 2%   （从工资中代扣）
```

### 4.5 个人账户与统筹

- 在职职工**个人缴费 2% 全额计入个人账户**；
- 单位缴费部分按规定比例划入个人账户（2021年门诊共济改革后划入比例下调，大部分划入统筹）；
- 个人账户用于支付门诊、定点药店购药、家庭共济等。

### 4.6 生育保险（合并后）

- 单位缴费**已含**生育保险（无单独项目），个人不缴；
- **生育津贴**：女职工产假期间，按单位上年度月平均工资 ÷ 30 × 产假天数发放（由生育保险基金支付）；
- **产假天数**：国家规定 98 天 + 各地延长生育假（上海 60 天、北京 60 天、广东 80 天等）；
- 产前检查费、生育医疗费由医保统筹支付。

## 五、失业保险

### 5.1 法律依据

- 《社会保险法》第四章；
- 《**失业保险条例**》（国务院令第258号）；
- 自 2017 年起，多省阶段性降低费率；自 **2023.5—2025.4** 阶段性降费政策延续（综合费率维持 1%）。

### 5.2 缴费比例

| 缴费主体 | 比例（示例：上海） | 备注 |
|---|---|---|
| **单位** | **0.5%** | 多数地区 0.5%；深圳 0.7% |
| **个人** | **0.5%** | 多数地区 0.5%；深圳 0.3% |

### 5.3 缴费基数

同养老保险基数。

### 5.4 计算

```
失业（单位）= 缴费基数 × 0.5%
失业（个人）= 缴费基数 × 0.5%   （从工资中代扣）
```

### 5.5 待遇

- 累计缴费 ≥ **1年**、非因本人意愿中断就业、已办理失业登记并求职 —— 可领取失业金；
- 领取期限按缴费年限分档（最长 24 个月）；
- 失业金月标准 = 当地最低工资标准 × 70%-90%（各地不同）。

## 六、工伤保险

### 6.1 法律依据

- 《社会保险法》第五章；
- 《**工伤保险条例**》（国务院令第586号，2010 修订）；
- 《**关于工伤保险费率问题的通知**》（人社部发〔2015〕71号）—— 八大行业风险类别。

### 6.2 缴费比例 — 八大行业风险类别

**单位单独缴费，个人不缴。**基准费率按行业风险类别浮动：

| 行业风险类别 | 基准费率 | 行业举例 |
|---|---|---|
| **一类**（风险较小） | **0.2%**（部分地区降至 0.16%） | 金融、IT、咨询、零售、教育、公共管理 |
| **二类** | **0.4%** | 房地产、租赁商务服务 |
| **三类** | **0.7%** | 农副食品加工、纺织、电气机械 |
| **四类** | **0.9%** | 通用设备制造、金属制品、汽车制造 |
| **五类** | **1.1%** | 燃气、化学原料、医药制造 |
| **六类** | **1.3%** | 黑色金属冶炼、煤炭洗选 |
| **七类** | **1.6%** | 石油天然气开采、煤炭采选 |
| **八类**（风险极大） | **1.9%**（部分地区 1.52%） | 有色金属采选、煤矿井下作业、烟花爆竹 |

> **浮动机制：** 根据单位上年度工伤发生率、工伤保险费使用情况，可在基准费率上下浮动 50%、20% 或维持基准。

### 6.3 计算

```
工伤（单位）= 缴费基数 × 行业基准费率（× 浮动系数）
工伤（个人）= 0
```

### 6.4 待遇

- 工伤医疗费、停工留薪期工资、一次性伤残补助金、伤残津贴、工亡补助金等；
- 工伤认定由参保地**人力资源和社会保障行政部门**作出；
- 劳动能力鉴定由**劳动能力鉴定委员会**作出。

## 七、住房公积金

### 7.1 法律依据

- 《**住房公积金管理条例**》（国务院令第350号，2002 颁布，2019 修订）；
- 各市《住房公积金管理办法 / 实施细则》。

### 7.2 缴费比例

| 缴费主体 | 比例 | 备注 |
|---|---|---|
| **单位** | **5% — 12%**（单位在区间内自选） | 单位与个人**比例必须一致** |
| **个人** | **5% — 12%**（与单位同比例） | 从工资中代扣 |

- 上海常见 **7% / 7%**；北京、杭州常见 **12% / 12%**；
- 单位可另设**补充公积金**（1%-5%），常见于国企、外企；
- 个人比例上限合并补充后通常不超过**24%**。

### 7.3 缴存基数

- **基数 = 员工上一年度月平均工资**（口径与社保相近但不完全一致 —— 公积金口径以"工资总额"为主，但部分地区在外籍员工、跨省工资上有差异）；
- **下限 = 当地最低工资标准**；
- **上限 = 上年度当地职工月平均工资 × 3**；
- 上海 2024.7—2025.6 上限约 **¥36,549/月**（与社保口径同步，但不绝对一致）；
- 每年 **7月1日**调整。

### 7.4 计算

```
公积金（单位）= 缴存基数 × 单位比例
公积金（个人）= 缴存基数 × 个人比例   （从工资中代扣）
合计存入员工公积金账户 = 单位部分 + 个人部分
```

> **关键**：公积金**全额计入员工个人公积金账户**，归员工所有，可用于购房贷款、租房提取、还贷提取、销户提取。

### 7.5 缴存与封存

- **新员工**：入职次月起缴存；
- **离职**：单位办理"封存"或"转移"；
- **跨地区流动**：公积金可全国转移接续。

## 八、缴费基数（上下限、各地差异）

### 8.1 通用原则

```
缴费基数 = 员工上一年度月平均工资  （工资总额口径）
若 > 上限  → 取 上限
若 < 下限  → 取 下限
新入职员工：以入职首月工资为初始基数；次年7月按全年口径重定
```

### 8.2 上下限来源

- **社保上限 = 上一年度当地全口径城镇单位就业人员平均工资 × 300%**；
- **社保下限 = 上一年度当地全口径城镇单位就业人员平均工资 × 60%**；
- **公积金上下限**由当地住房公积金管理委员会单独公告，**口径与社保接近但不绝对相同**。

### 8.3 调整时点

| 城市 | 社保基数调整月 | 公积金基数调整月 |
|---|---|---|
| 上海 | 每年 7月 | 每年 7月 |
| 北京 | 每年 7月 | 每年 7月 |
| 深圳 | 每年 1月（部分险种）/ 7月 | 每年 7月 |
| 广州 | 每年 7月 | 每年 7月 |
| 杭州 | 每年 1月 | 每年 7月 |

### 8.4 五险一金基数是否同步？

- **多数城市社保基数与公积金基数同步**（口径相同、上下限相同）；
- **部分城市分立**（深圳、苏州等地公积金口径单独公告）；
- 单位**不得**通过工资拆分（如"工资 + 报销"）规避缴费基数 —— 2019.1.1 起社保由**税务部门统一征收**，**金税四期**已实现工资数据与社保基数交叉比对。

### 8.5 常见低报风险

| 低报情形 | 监管反应 |
|---|---|
| 一律按最低基数缴 | 税务/社保稽核后**追补 + 滞纳金（日万分之五）+ 1-3 倍罚款** |
| 工资拆为"工资 + 报销" | 报销部分计入工资总额，需补缴 |
| 实习生 / 非全日制 / 劳务关系误判 | 重新认定为劳动关系后，**追溯参保** |
| 跨省用工归属错配 | 用工地稽核认定补缴 |

## 九、外籍员工社保

### 9.1 法律依据

- 《**在中国境内就业的外国人参加社会保险暂行办法**》（人社部令第16号，**自 2011.10.15 施行**）；
- 双边/多边**社会保险协定**（截至2025年，中国已与德国、韩国、丹麦、芬兰、加拿大、瑞士、荷兰、西班牙、卢森堡、日本、塞尔维亚、法国 等 12+ 国签署）。

### 9.2 原则

- **在中国境内就业的外国人**（持工作许可与居留许可）原则上应参加 **基本养老、基本医疗、失业、工伤、生育** 五险；
- 比例与基数与中国公民相同；
- 离境时可申请退还**个人账户**余额或保留待遇。

### 9.3 实务差异

- **上海**长期以来执行宽松，事实上对外籍员工社保未强制；2025年起执法逐步收紧；
- **北京、深圳、广州**总体按规定强制参保；
- 实际操作前**必须**核对**参保地最新口径**。

### 9.4 社保协定豁免

| 协定国 | 可豁免险种 | 豁免期限 |
|---|---|---|
| 德国 | 养老、失业 | 派遣期内（通常 ≤ 48 个月，可延长） |
| 韩国 | 养老、失业 | 派遣期内 |
| 日本 | 养老 | 派遣期内 |
| 加拿大 | 养老 | 派遣期内 |
| 荷兰、西班牙、卢森堡、瑞士、芬兰、丹麦、塞尔维亚、法国 | 视协定条款（多为养老） | 派遣期内 |

> **豁免凭证：** 来华前由派出国主管机构出具"参保证明"（如德国 A1 / Z 表），由用人单位在中国社保经办机构办理"免缴登记"。

### 9.5 公积金

- 《住房公积金管理条例》未将外籍员工纳入强制范围；
- **上海、北京**等地允许外籍员工**自愿**缴存公积金；
- **深圳**等地外籍员工不允许缴存。

### 9.6 港澳台员工

- 自 **2020.1.1** 起（人社部令第41号废止 2005 年第26号令），港澳台居民在内地（大陆）就业**应当依法参加社会保险**；
- 与外籍员工口径基本一致；但港澳台不适用社保协定豁免（无双边协定，因属"一国")。

## 十、计算示例（上海某软件工程师月薪 3 万元）

**场景：** 软件工程师，户籍上海，2025 年 3 月在某上海互联网公司任职，月工资总额 **¥30,000**（含基本工资、绩效、加班费）。单位为一类风险行业（IT）。

**假设输入（必须以当地最新公告核实）：**

- 缴费基数 = ¥30,000/月（在上海上下限 ¥7,310 — ¥36,549 区间内，按工资总额计）
- 养老：单位 16% / 个人 8%
- 医疗：单位 9.5% / 个人 2%
- 失业：单位 0.5% / 个人 0.5%
- 工伤：单位 0.16%（一类）
- 公积金：单位 7% / 个人 7%

### 10.1 单位与个人缴费明细

| 项目 | 单位（元/月） | 个人（元/月） |
|---|---|---|
| 养老（16% / 8%） | 4,800.00 | 2,400.00 |
| 医疗（9.5% / 2%） | 2,850.00 | 600.00 |
| 失业（0.5% / 0.5%） | 150.00 | 150.00 |
| 工伤（0.16% / —） | 48.00 | 0.00 |
| 住房公积金（7% / 7%） | 2,100.00 | 2,100.00 |
| **合计** | **9,948.00** | **5,250.00** |

### 10.2 个税专项扣除衔接（详见 `china-pit`）

```
工资总额                            30,000.00
减：个人五险一金（专项扣除）        (5,250.00)
减：起征点（5,000/月）              (5,000.00)
减：专项附加扣除（假设 1,500，由 china-pit 处理） (1,500.00)
应纳税所得额（月度累计预扣口径需另算） = 18,250.00
```

> 月度累计预扣预缴的实际算法见 `china-pit`，本技能只输出"个人五险一金 = ¥5,250"作为专项扣除项。

## 十一、申报与缴纳

### 11.1 月度节奏

| 事项 | 缴款对象 | 期限 | 渠道 |
|---|---|---|---|
| **社保费申报缴费**（养老、医疗、失业、工伤；含个人 + 单位部分） | 主管税务机关（**自 2019.1.1 起社保由税务部门统一征收**） | **次月 15 日前**（部分地区 5-25 日不等） | 当地**电子税务局** / 社保经办系统 |
| **住房公积金汇缴** | 当地住房公积金管理中心 | **次月 15 日前**（部分地区 5 日或当月内） | 当地**住房公积金网厅** + 银行批扣 |
| **个税专项扣除联动** | — | 与个税申报同步 | 通过 `china-pit` 编排 |

### 11.2 年度节奏

| 事项 | 期限 | 备注 |
|---|---|---|
| **社保缴费基数申报**（年度核定） | **每年 6—7 月**（上海口径） | 单位申报员工上一年度月平均工资，作为本年度缴费基数 |
| **公积金缴存基数调整** | **每年 6—7 月** | 单位申报员工上一年度月平均工资 |
| **工伤行业基准费率浮动核定** | 通常年初 | 由人社部门按工伤发生率核定下浮 / 上浮 |
| **年度个税汇算清缴** | **次年 3.1—6.30** | 详见 `china-pit` |

### 11.3 罚则

| 违规 | 后果 |
|---|---|
| 未按时申报社保 | 每日**万分之五**滞纳金 + 1-3 倍罚款（《社会保险法》第86条） |
| 未按时缴纳社保 | 同上 + 列入失信名单 + 影响投标资质 |
| 少缴 / 漏缴公积金 | 责令限期补缴；逾期可申请法院强制执行（《住房公积金管理条例》第37条） |
| 未办理工伤认定缴费 | 工伤事故发生后由单位承担全部工伤待遇 |
| 拒不参保（外籍员工） | 责令整改 + 罚款；金税四期+人社联网核查后追溯 |

## policyengine-canada

- Repository: [PolicyEngine/policyengine-canada](https://github.com/PolicyEngine/policyengine-canada)
- License: AGPL-3.0
- Stars: 17
- Language: Python
- Scope: Country model of Canada's tax-benefit system using PolicyEngine Core. Covers federal and provincial income tax, GST/HST credit, Canada Child Benefit, CPP/EI contributions.
- Integration: Same AGPL-3.0 license — fully compatible. Tax-benefit calculation logic and rate tables can be directly incorporated.

# Argentina — Related Open-Source Projects

OpenAccountants is AGPL-3.0. AGPL-3.0 and LGPL-3.0 are compatible licenses. Projects below can be incorporated with attribution.

## pyafipws

- Repository: [reingart/pyafipws](https://github.com/reingart/pyafipws)
- License: LGPL-3.0
- Stars: 353
- Language: Python
- Scope: Factura Electrónica AFIP y otros servicios web. Interfaces, tools and apps for Argentina's government webservices (SOAP, COM/DLL, PDF, DBF, XML, JSON). The definitive open-source Argentine e-invoicing library.
- Integration: LGPL-3.0 — compatible. AFIP web service integration patterns, e-invoice generation logic, and authentication flows directly reusable for Argentine tax compliance automation.

## PyARCA

- Repository: [GeraCollante/PyARCA](https://github.com/GeraCollante/PyARCA)
- License: LGPL-3.0
- Stars: 9
- Language: Python
- Scope: CLI para facturación electrónica Monotributo (ARCA/ex-AFIP). Fork of pyafipws focused on Monotributo invoicing.
- Integration: LGPL-3.0 — compatible. Reference for Monotributo-specific invoicing workflows and ARCA (formerly AFIP) API interactions. Useful for validating Monotributo compliance logic.
